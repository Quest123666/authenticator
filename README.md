## 📌 项目说明
这是一个纯前端、完全离线可用的 TOTP 二次验证码工具，部署在 GitHub Pages 上，任何设备打开网址即可直接使用，无需安装任何客户端。

### 适配修改说明
基于原项目 `v1.0.0` 版本的构建产物，仅做了以下修改，确保完全离线可用：
- 移除了外部 `Google Fonts` 字体引用，替换为系统无衬线字体，消除外部网络依赖
- 未修改任何验证码生成、账号导入等核心逻辑，功能与原项目完全一致

---

## ✨ 功能特点
- **纯前端离线运行**：所有计算在本地浏览器完成，无后端、无数据上传，隐私安全有保障
- **多种账号导入方式**：
  - 直接输入 2FA Secret 密钥
  - 粘贴 `otpauth://` 格式链接自动解析账号信息
  - 支持摄像头扫码/图片识别导入二维码
- **便捷操作**：点击验证码一键复制到剪贴板，自动刷新
- **深色/浅色模式切换**：适配不同使用场景
- **账号备份导出**：生成二维码快速备份或分享账号

---

## 🚀 使用方法
1.  访问部署地址：`https://quest123666.github.io/authenticator/`
2.  导入你的 2FA 账号：
    - 方式1：直接在输入框粘贴你的 2FA Secret 密钥
    - 方式2：粘贴完整的 `otpauth://` 链接，页面会自动解析账号信息
    - 方式3：点击相机/图片图标，扫码或上传二维码导入
3.  页面会立即生成动态 6 位验证码，每 30 秒自动刷新一次
4.  点击验证码即可一键复制到剪贴板，直接粘贴到需要验证的登录页面即可

---

## 📌 注意事项
- 所有数据仅保存在浏览器本地缓存中，建议定期备份你的 2FA Secret 密钥，避免丢失
- 为了你的账号安全，请勿在公共设备上保存或使用该工具
- 本工具仅支持标准 TOTP 协议，不支持 HOTP 协议
- 项目为个人练手部署，无任何维护与技术支持，建议定期参考原项目更新
